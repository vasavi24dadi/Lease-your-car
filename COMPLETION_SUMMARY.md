## 🚀 How to Run Your Application

### Quick Start (3 Steps):

**Step 1 - Terminal 1 (Backend):**
```bash
cd backend
node server.js
```
⏳ Wait for: "Server started on http://localhost:5000"

**Step 2 - Terminal 2 (Frontend):**
```bash
cd frontend
npm start
```
⏳ Wait for: Browser opens http://localhost:3000

**Step 3 - Test Features:**
- Create Owner account
- Add car with image
- Create Customer account
- Browse and book car

## 🗄️ Database Setup

### Step-by-step:

1. **Create Database:**
   ```sql
   CREATE DATABASE lease_your_car;
   ```

2. **Run Schema:**
   ```bash
   psql -U postgres -d lease_your_car -f backend/database_schema.sql
   ```

3. **Tables Created:**
   - users (with role field)
   - cars (uploaded by owners)
   - bookings (customer bookings)
   - messages (chat history)
   - ratings (reviews)
